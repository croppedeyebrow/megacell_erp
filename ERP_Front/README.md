# ERP_Front

MegaCell ERP 웹 UI (React + TypeScript + Vite).

## 실행

```bash
cd ERP_Front
npm install
npm run dev
```

http://localhost:5173

데모 로그인: 아무 이메일 + 4자 이상 비밀번호

## 내부 구조

```text
src/
  auth/            인증
  components/      ui / layout / data
  features/        업무 화면
  navigation/      메뉴
  mocks/           UI mock
  styles/          디자인 토큰
  routes/          라우팅
  lib/             format, status helpers
```
