# MegaCell ERP 실행 안정화 안내

## 실행

`start_erp.bat`를 더블클릭합니다.

- 서버가 꺼져 있으면 Streamlit 서버를 실행합니다.
- 서버가 이미 켜져 있으면 브라우저만 엽니다.
- 직원용 사내 접속 주소도 같이 표시합니다.

## 종료

`stop_erp.bat`를 더블클릭합니다.

## 상태 확인

`erp_status.bat`를 더블클릭합니다.

## 직원 접속

같은 사내 네트워크에 있는 직원은 아래 형태로 접속합니다.

```text
http://서버PC_IP:8501
```

예:

```text
http://172.30.1.71:8501
```

## 방화벽 허용

직원 PC에서 접속이 안 되면 서버 PC에서 `firewall_allow_8501_admin.bat`를 관리자 권한으로 실행합니다.

## 주의

- ERP 서버 PC가 켜져 있어야 합니다.
- `MegaCell ERP Server` 창을 닫으면 ERP 접속이 끊깁니다.
- 외부 인터넷 공개는 아직 권장하지 않습니다. 외부 접속은 로그인/HTTPS/권한 설계 후 진행하는 것이 안전합니다.
